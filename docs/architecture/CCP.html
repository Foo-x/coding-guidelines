# 閉鎖性共通の原則

CCP (Common Closure Principle)
パッケージに含まれるクラスは、みな同じ種類の変更に対して閉じているべきである

パッケージの変更理由は1つだけにしましょう。また、パッケージ内の変更が他のパッケージに影響を及ぼさないようにしましょう。

NOTE

パッケージが変更される理由は1つだけにする
