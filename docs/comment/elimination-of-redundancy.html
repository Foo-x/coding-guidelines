# 冗長性の排除

コメントとして価値のあることだけを書きましょう。コメントがあるとその分だけ読むのに時間がかかるため、価値のないものは消すべきです。

価値のないコメントとは以下のようなものです。

  • 名前、引数、戻り値など、宣言からわかること
  • 実装者のためのコメントの場合、コードを読めばすぐにわかること
    • 理解するのに時間がかかりそうな場合はコメントすべきです

NOTE

価値のないコメントを書かない
