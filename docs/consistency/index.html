# 美しさ

美しいコードの基本は一貫性があることです。一貫性があるコードには以下のメリットがあります。

  • 読みやすい
  • 理解しやすい
  • 変更を加える際に目的の箇所をすぐに見つけられる

NOTE

コードに一貫性を持たせる
