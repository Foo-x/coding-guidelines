    # 条件式の並び順

    条件式は左側に「調査対象」、右側に「比較対象」を置きましょう。英語の文法と同じ順番にすることで読みやすくなります。

    # ❌

    if (expected == actual)

    # ⭕️

    if (actual == expected)

    ✏️ 条件式は左側に「調査対象」、右側に「比較対象」
