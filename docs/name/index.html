# 情報量の最大化

名前に含まれる情報量を最大化しましょう。名前はコメントの一部です。適切に命名された変数、メソッド、クラスは、中を見なくてもその役割を理解できます。命名は「読む人が最短で理解できるコード」として最も重要な要素の一つです。
