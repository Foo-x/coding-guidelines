先入観

読んだ人に誤解されない名前をつけましょう。特定の場面でよく使われる単語は先入観を持たれやすいです。たとえば、getはgetterとして、sizeやlengthはリストの長さとしてよく使われます。これらは軽い処理であると思われやすいので、重い処理を行わないようにしましょう。重い処理を行う場合は別の名前にすべきです。
